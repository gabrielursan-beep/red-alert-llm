RED ALERT LLM

Manual de Urgenta — Asistent AI Offline pe SSD USB

Ce este?

Red Alert LLM este un kit AI portabil care functioneaza FARA internet. SSD-ul contine:

  • Un asistent AI inteligent — vorbeste romana si alte 118 limbi
  • Wikipedia completa — romana + engleza + enciclopedie medicala
  • Ghiduri practice — reparatii electronice, calatorii, bricolaj
  • Totul gata configurat — doar conectati SSD-ul si porniti

Ce aveti nevoie

  • SSD-ul USB (acesta pe care l-ati primit)
  • Un computer cu minim 8 GB RAM (Windows sau Mac)
  • SAU un telefon Android/iPhone cu minim 6 GB RAM
  • Port USB liber (USB-A clasic sau USB-C)
REGULA DE AUR — Scoateti SSD-ul IN SIGURANTA!
Windows: click dreapta pe SSD in "Acest PC" → Eject
macOS: trageti SSD-ul in Cos (Trash) sau click dreapta → Eject
Telefon: notificare "Eject" inainte de deconectare

NU scoateti SSD-ul cat timp AI-ul sau Wikipedia ruleaza!
Opriti mai intai (vedeti instructiunile de oprire pe fiecare pagina).

Ce gasiti pe SSD

  • Asistent AI — 3 modele de inteligenta artificiala (mic/mediu/mare)
  • Wikipedia Engleza — 6.8 milioane articole (doar text, 48 GB)
  • Wikipedia Romana — completa cu imagini (10.6 GB)
  • Enciclopedie Medicala — 73.000+ articole medicale
  • Ghiduri reparatii (iFixit), Calatorii (Wikivoyage), Dictionar (Wiktionary)

Fisierul magic: START-HERE.html

Deschideti fisierul START-HERE.html de pe SSD in orice browser pentru un ghid interactiv care detecteaza automat sistemul dvs.

Pe Laptop Windows

Windows 10 sau Windows 11

A. Pornire Asistent AI

  1. Conectati SSD-ul la portul USB al laptopului
  2. Deschideti SSD-ul in "Acest PC" (sau "This PC")
    Gasiti folderul red-alert-llm
  3. Deschideti folderul launchers
  4. Dublu-click pe start-windows.bat
Daca apare un ecran albastru de protectie (SmartScreen):
Click "More info" (Mai multe informatii) → apoi "Run anyway" (Ruleaza oricum)
  1. In fereastra neagra, tastati 1 si apasati Enter
    (alegeti llamafile — varianta simpla)
  2. Asteptati 10-30 secunde — browserul se deschide automat
    Daca nu se deschide, tastati in browser: http://localhost:8080
  3. Scrieti intrebarea in caseta de text si apasati Enter
    AI-ul raspunde! Puteti vorbi in romana sau engleza.

B. Pornire Wikipedia Offline

  1. Din folderul launchers, dublu-click pe start-kiwix-windows.bat
  2. Se deschide Kiwix — programul de Wikipedia offline
  3. Din Kiwix: File → Open File → navigati la SSD → knowledge → alegeti un fisier .zim
  4. Cautati orice articol, exact ca pe Wikipedia!
Prima data: Daca Kiwix nu este pe SSD, descarcati-l gratuit de la kiwix.org
Extrageti in folderul apps\kiwix\kiwix-windows\ de pe SSD.
Oprire:
• AI: inchideti fereastra neagra (sau apasati Ctrl+C in ea)
• Wikipedia: inchideti Kiwix normal (X)
• Scoateti SSD-ul IN SIGURANTA (click dreapta → Eject)

Pe Laptop macOS

MacBook, iMac, Mac Mini (Apple Silicon sau Intel)

A. Pornire Asistent AI

  1. Conectati SSD-ul la portul USB-C
    (sau USB-A cu adaptor daca aveti Mac mai vechi)
  2. Deschideti Finder → in bara laterala, gasiti SSD-ul
    Navigati la red-alert-llm → launchers
  3. Dublu-click pe start-macos.command
Daca macOS spune "nu poate fi deschis" (Gatekeeper):
1. Click DREAPTA pe fisier → Open → click "Open" din nou
2. SAU: System Settings → Privacy & Security → scroll jos → "Open Anyway"
Acest lucru se face o singura data.
  1. Se deschide Terminal-ul. Tastati 1 si apasati Enter
    (alegeti llamafile — varianta simpla)
  2. Asteptati 10-30 secunde — browserul se deschide automat
    Daca nu: deschideti Safari si tastati http://localhost:8080
  3. Scrieti intrebarea si apasati Enter — AI-ul raspunde!

B. Pornire Wikipedia Offline

  1. Instalati Kiwix din App Store (o singura data, gratuit):
    App Store → cautati "Kiwix" → Install
  2. Deschideti Kiwix → File → Open File
  3. Navigati la SSD → knowledge → alegeti un fisier .zim
  4. Cautati articole exact ca pe Wikipedia!
Oprire:
• AI: inchideti fereastra Terminal (sau apasati Ctrl+C)
• Wikipedia: inchideti Kiwix normal (Cmd+Q)
• Scoateti SSD-ul: click dreapta pe SSD in Finder → Eject
  SAU trageti iconita SSD-ului in Cos (Trash) din Dock
Performanta: Pe MacBook cu Apple Silicon (M1, M2, M3, M4), AI-ul raspunde de 3-5 ori mai rapid decat pe un laptop Windows obisnuit. Raspunsurile vin aproape instant.

Pe Telefon

ANDROID

Aveti nevoie de: telefon cu 6+ GB RAM, port USB-C (sau adaptor OTG)

Asistent AI:

  1. Conectati SSD-ul la telefon prin USB-C
  2. Instalati "PocketPal AI" din Google Play Store (gratuit)
  3. Deschideti File Manager → SSD → models → mobile
  4. Copiati qwen3-4b-q3_k_m.gguf pe telefon
    (~2 GB, dureaza 2-3 min)
  5. Deschideti PocketPal AI → Import Model → selectati fisierul copiat
  6. Asteptati 30-60 sec — vorbiti cu AI-ul!

Wikipedia:

  1. Din File Manager: SSD → apps → android
  2. Instalati kiwix-android-full.apk
    (NU versiunea din Play Store)
  3. Deschideti Kiwix → SSD → knowledge → alegeti un .zim

iPHONE / iPAD

Aveti nevoie de: iPhone 15+ (USB-C) sau adaptor Lightning, 6+ GB RAM

Asistent AI:

  1. Conectati SSD-ul la iPhone prin USB-C
  2. Instalati "PocketPal AI" din App Store (gratuit)
  3. Deschideti Files (Fisiere) → gasiti SSD-ul → models → mobile
  4. Tineti apasat pe qwen3-4b-q3_k_m.gguf → Copy
    Navigati la "On My iPhone" → Paste
    (~2 GB, tineti ecranul aprins!)
  5. Deschideti PocketPal AI → Import from Files → selectati fisierul
  6. Asteptati 30-60 sec — vorbiti cu AI-ul!

Wikipedia:

  1. Instalati Kiwix din App Store (gratuit)
  2. Din Files, copiati un .zim mic pe iPhone
    (Wikipedia RO ~2.3 GB)
  3. Kiwix → Open File → selectati .zim
Limitari importante pe telefon:
• Telefonul se incalzeste in timpul utilizarii AI — este normal
• Bateria se consuma rapid — tineti telefonul la incarcat
• Pe iPhone, modelul TREBUIE copiat pe telefon (nu merge direct de pe SSD)
• Telefoanele cu 4 GB RAM pot avea dificultati — inchideti alte aplicatii
• Wikipedia EN completa (48 GB) NU incape pe telefon — folositi Wikipedia RO (2.3 GB)
