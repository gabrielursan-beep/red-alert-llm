# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2026-03-24

### Added
- AI Chat engine: llamafile v0.10.0 + KoboldCpp v1.110
- AI Models: Qwen3 (4B/8B/14B), Gemma-3-4B, mobile Q3_K_M variant
- Offline knowledge: 9 ZIM files (Wikipedia EN/RO, WikiMed, iFixit, Wiktionary, Wikivoyage, StackExchange)
- Encrypted vault support via VeraCrypt (200 GB)
- Cross-platform launchers: Windows (.bat), macOS (.command), Linux (.sh)
- Auto RAM detection selects optimal model (4B/8B/14B)
- Auto GPU detection (NVIDIA CUDA, Apple Metal)
- Setup scripts with auto-download for all platforms
- Interactive knowledge base downloader with 3 profiles (minimal/recommended/full)
- START-HERE.html: self-contained gateway with OS detection
- MANUAL-URGENTA-RO.html: 4-page printable emergency manual in Romanian
- Bilingual documentation (English + Romanian)
- Verification scripts (verify-setup.bat/.sh)
- Kiwix Desktop 2.5.1 portable for Windows
